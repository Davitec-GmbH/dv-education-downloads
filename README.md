# Education Downloads (dv_education_downloads)

Download center for the education sector with document type, language, version, validity period, and filtering.

## Features

- Download management with document type categorization
- Multi-language support (DE/EN)
- Version tracking
- Validity period (valid from/until)
- Search and multi-filter (type, language)
- Table view with download links
- TYPO3 v12 LTS and v13 LTS support

## Installation

```bash
composer require davitec/dv-education-downloads
```
